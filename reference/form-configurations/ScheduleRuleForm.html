    Can be 'recurring', 'specific', or 'range'; if 'recurring' the property 'week_days' will be used to determine the actual schedule rule dates; if 'specific', the user_dates array will be used to determine the schedule rule dates; if 'range', then user_date_start and user_date_end will be used to determine the actual schedule rule (range).
  • #run_hours_mode ⇒ String
