---
title: Categories
permalink: /reference/categories/
---
You can think of Categories as a specialized custom attribute - it should be used if you have tree structure data, for example:

```
Industry
├─ Industry 1
│  ├─ Specialization 1
|  |  ├─ Sub Specialization 1
|  |  └─ Sub Specialization 2
|  └─ Specialization 2
└─ Industry 2
```
